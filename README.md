# 🚀 AlgoDarshan

An interactive web application to visualize **Sorting**, **Searching**, and **Pathfinding Algorithms** in real time with animated execution and algorithm insights.

🌐 **Live Demo:**  
👉 https://algorithm-visualizer-azure-eta.vercel.app/

---

## 📌 Features

### 🔢 Sorting Algorithms
- Bubble Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Heap Sort

✅ Dynamic array generation  
✅ Speed control  
✅ Active & sorted element highlighting  

---

### 🔍 Searching Algorithms
- Linear Search
- Binary Search

✅ Step-by-step visualization  
✅ Target tracking  
✅ Complexity explanation panel  

---

### 🧭 Pathfinding Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra Algorithm

✅ Grid-based visualization  
✅ Wall / obstacle creation  
✅ Shortest path reconstruction  
✅ Animated traversal  
✅ Algorithm information panel  

---

## 🧠 Algorithms Visualized

| Category | Algorithms |
|----------|-----------|
| Sorting | Bubble, Selection, Merge, Quick, Heap |

## 🎨 Visualization Legend

| Color | Meaning |
|------|---------|
| 🟩 Green | Start Node |
| 🟥 Red | End Node |
| ⬛ Black | Wall / Obstacle |
| 🟦 Blue | Visited Node |
| 🟨 Yellow | Shortest Path |

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/SKEL1NJA/AlgoDarshan.git
```

Navigate into project:

```bash
cd AlgoDarshan
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:
```bash
http://localhost:5173
```
